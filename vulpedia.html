Eerste (vanaf nul) Webontwerp Project

Dit was mijn eerste website-project. Je kunt de broncode hier vinden: https://github.com/AliAl-Gburi/Vulpedia

De website ging over vossen en hun ondersoorten. Het was een soort eenvoudige encyclopedie over vossen. De inhoud van de website is niet relevant, het was een excuus om een website te maken.

The steps towards make the website went as follows:

  • Maak de nodige HTML-bestanden om de website een betekenisvolle structuur te geven
  • Voeg een favicon en wat styling toe
  • Maak de website responsive
  • Gebruik flex en grids om de website te stylen

Als ik opmerkelijke delen in de code van de website zou willen specificeren, zou ik zeggen dat het het gebruik van grids in mijn css-bestand is:

#contacts p {
display: grid;
grid-template-areas: "label inputfield";
grid-template-columns: 2fr 4fr;
grid-column-gap: 1em;
align-items: center;
}
label, .radiolabel {
grid-area: label;
grid-column-start: 1;
text-align: right;
}
#firstname, #lastname, #email, input, textarea, .error, #browser {
